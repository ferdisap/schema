S1000D XML Schema (Issue 5.0)

Overview of available schema files under xml_schema_flat

The following list includes the core XML Schema Definition (.xsd) files published as part of the S1000D Issue 5.0 under the xml_schema_flat directory. These schemas define the structure and validation rules for various S1000D data modules and supporting files.

Core Schema Files

  • brdocXsd - Business Rules definition.
  • brexXsd - Business Rules Exchange schema.
  • checklistXsd - Checklist document.
  • commentXsd - Commenting CSDB Object.
  • comrepXsd - Common repository document
  • containerXsd
  • crewXsd - Crew/operator data module schema.
  • ddnXsd - Data Dispacth Note schema.
  • descriptXsd - Descriptive data module schema.
  • dmlXsd - Data Module List schema.
  • faultXsd - Fault data module schema.
  • frontmatterXsd - Front matter data module schema.
  • icnmetadataXsd - Illustration Control Number schema.
  • ipdXsd - Illustrated Parts Data schema.
  • learningXsd - Learning data module schema.
  • pm.xsXsd - Publication Module schema.
  • procedXsd - Procedural data module schema.
  • processXsd - Process schema.
  • rdfXsd - RDF schema
  • sbXsd - Service Bulleting schema.
  • schedulXsd - Scheduling schema.
  • scocontentXsd - SCORM content schema.
  • scormcontentpackageXsd - SCROM content package schema.
  • wrngdataXsd - Wiring data schema.
  • wrngfldsXsd - Wiring fields schema.

Supporting and Reference Schemas

  • appliccrossreftableXsd - Applicability product schema.
  • condcrossreftableXsd - Condition applicability product schema.
  • prdcrossreftableXsd - Certain product schema
  • dcXsd - Dublic Core schema.
  • updateXsd - Informing the update CSDB object (manuals).
  • xcfXsd - Webcgm schema.
  • xlinkXsd - Cross link schema.
