S1000D XML Schema (Issue 5.0)

Overview of available schema files under xml_schema_flat

The following list includes the core XML Schema Definition (.xsd) files published as part of the S1000D Issue 5.0 under the xml_schema_flat directory. These schemas define the structure and validation rules for various S1000D data modules and supporting files.

Core Schema Files

  • descript.xsd — Descriptive data module schema.
  • proced.xsd — Procedural data module schema.
  • crew.xsd — Crew/operator data module schema.
  • pm.xsd — Publication Module schema.
  • dmrl.xsd — Data Module Requirements List schema.
  • icn.xsd — Illustration Control Number schema.
  • pmEntry.xsd — Publication Module Entry schema.
  • brepm.xsd — Business Rules Exchange schema.
  • ipd.xsd — Illustrated Parts Data schema.
  • learn.xsd — Learning data module schema.

Supporting and Reference Schemas

  • common.xsd — Common data structures and shared definitions.
  • content.xsd — Generic XML content and media definitions.
  • ident.xsd — Identification and metadata schema.
  • xlink.xsd — XLink schema used for linking and references.
  • dmIdent.xsd — Data module identification schema.
  • schematron.xsd — Validation schema for business rules.
