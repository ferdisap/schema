S1000D XML Schema (Issue 5.0)

Public Information Page

The S1000D specification defines a structured and standardized way of producing and exchanging technical publications for equipment maintenance and operation. As part of the standard, the S1000D community provides official XML Schema Definitions (XSD) that describe the structure and validation rules of compliant XML data modules and publications.

The official schemas are available at: http://www.s1000d.org/S1000D_5-0/

Available Schema Directories

Visit this site to get flat schema https://ferdisap.github.io/schema/flat.html
  • ent — For electronic navigation chart.
  • lom_schema — Learning Object Metadata schema used for training content.
  • notations — Contains XML notation definitions used within the standard.
  • xml_schema_cat — Catalog of schema references and relationships.
  • xml_schema_flat — Flattened XML Schemas for validation and development purposes.

Purpose of These Schemas

These schema files define the data structures, attributes, and validation rules used across all S1000D-compliant XML documents. They are intended for:

  • Software developers building S1000D-compliant tools
  • Technical authors validating XML data modules
  • Organizations integrating content into S1000D-based documentation systems
